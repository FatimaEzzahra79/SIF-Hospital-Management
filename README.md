# SIF-Hospital-Management

## 📝 Description
This is my first web development project! **SIF-Hospital-Management** is a simple and clean Hospital Management System designed to help manage patient records and medical staff information efficiently.

## 🚀 Main Features
*   **Patient Management:** Tools to view and manage patient data.
*   **Modern UI:** A clean, user-friendly interface for hospital staff.
*   **Responsive Design:** Optimized for both desktop and mobile screens.

## 🛠️ Tech Stack
Based on the files in this repository:
*   **HTML5:** Structural layout of the application.
*   **CSS3:** Styling and custom layouts (see `hospital.css`).
*   **JavaScript:** Interactive elements and logic (see `hospital.js`).

## 💻 How to Run
1. Clone the repository to your machine.
2. Open `hospital.html` in any modern web browser.
